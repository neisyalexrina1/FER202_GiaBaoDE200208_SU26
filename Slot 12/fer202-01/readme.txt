Student Feedback & Course Review
--------------------------------

1. Installed packages:
- react, react-dom
- react-router-dom (for routing)
- axios (for API calls)
- bootstrap, react-bootstrap (for UI components)
- concurrently (to run frontend and backend simultaneously)
- json-server (dev dependency, to mock backend API)

2. How to run the application:
- Open terminal and navigate to the root directory of this project.
- Run `npm install` to install all required dependencies.
- Run `npm start` to start both the React app and JSON Server at the same time.
- Open your browser and go to `http://localhost:5173`.
- You can log in using the sample accounts provided in db.json (e.g., student01@fpt.edu.vn / 123456).
